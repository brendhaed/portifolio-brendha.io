<h2>🎯 Objetivo do projeto</h2>
<p>O portfólio foi desenvolvido para:</p>
<ul> 
<li>Apresentar meus projetos de forma clara e organizada</li>

<li>Modo dark/light</li>

<li>Animações suaves para melhorar a experiência do usuário</li>
</ul>
